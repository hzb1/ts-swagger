
本项目是一个 Swagger API 调试工具，主要特点：

* 用户在 网页中输入 Swagger 服务地址（IP / URL）（可选可输入框，具备缓存功能）
* 支持 Swagger v3（默认） / v2
* Web 页面本身不直接请求后端 API，而是通过 Chrome Extension 转发，解决浏览器跨域问题
* 和其它功能不同的是，接口文档是以 TS 类型展示（请求参数 / 响应结果）
* 页面主要职责：
  * 解析 Swagger 文档
  * 展示 API 结构
  * 生成请求参数
  * 展示请求与响应结果（TS 类型）
  * 可修改项目配置
    * 请求函数模板 (JS)
    * 生成选项：
      * 展示 Example（单选框，默认选中）
      * Int64转String （单选框，默认选中）
      * Interface（单选框，默认选中）
      * Export（单选框，默认选中）
      * 分号 （单选框，默认选中）

期待生成的前端代码技术栈：
* 前端：React 19 + TypeScript + Ant Design v6 + highlight.js

画布尺寸：支持响应式布局，适配不同屏幕尺寸，PC端优先
支持深浅主题

顶部固定 Header
Header 下方为左右两栏布局：
左侧 Sidebar（API 树）：宽度 280px
右侧主内容区：自适应宽度

### 三、顶部 Header 设计

* 左侧：
  * 应用标题文本：“TS Swagger UI”

* 中间区域：
  * Swagger 地址输入/选择框
  * Swagger 版本选择下拉框，选项：v3（默认）、v2
  * 主按钮：“加载 Swagger”

* 右侧：
  * 插件状态指示 Badge
    * 圆形，插件启用时为绿色，插件禁用时为灰色
  * 项目配置图标
    * 图标：齿轮形状
    * 悬停提示：“项目配置”
    * 点击触发项目配置弹窗
    * 弹窗内容：
      * 请求函数模板 (JS)
      * 生成选项：
        * 展示 Example（单选框，默认选中）
        * Int64转String （单选框，默认选中）
        * Interface（单选框，默认选中）
        * Export（单选框，默认选中）
        * 分号 （单选框，默认选中）
  * 搜索框：
    * 点击后打开搜索弹窗
    * 弹窗内容：
      * 搜索框：用于输入搜索关键词，Placeholder 为 '搜索路径/名称/类名...'
      * 搜索结果分组列表：展示符合条件的 API 项
        * 分组名称，滚动时保持当前分组可见
        * 每个 API 项显示 HTTP 方法 + API 路径 + 中文名称 + (匹配方式?：路径 | 名称 | 类名)
        * HTTP 方法颜色规范： GET：绿色 、POST：蓝色 、PUT：橙色 、DELETE：红色
      * 底部是快捷键提示（“↑” “↓” 导航选择项，“Enter” 选择当前项，“Esc” 关闭弹窗）
